# CampoSphere — Plugin QGIS

Plugin permettant d'extraire les bâtiments modulaires présents sur une liste de sites clients.

---

## Comment ça marche

### 1. Charger la couche de points (sites clients)

Deux possibilités :
- **"Accéder aux fichiers"** → charger un Shapefile ou Geopackage
- **"Géocoder CSV/XLS"** → charger un fichier CSV ou Excel, le plugin le convertit automatiquement en points

### 2. Charger la couche BM (bâtiments modulaires)

Cliquer sur **"Accéder aux fichiers"** → charger le Shapefile ou Geopackage des bâtiments modulaires.

### 3. Charger les parcelles cadastrales

Cliquer sur **"Accéder aux fichiers"** → charger le Shapefile des parcelles cadastrales du département.

> Téléchargeables gratuitement sur https://cadastre.data.gouv.fr

### 4. Nommer le fichier de sortie

Saisir un nom → cliquer sur **"Valider le nom"**.

### 5. Lancer le traitement

Cliquer sur **"Lancer le traitement"** → la couche résultat s'affiche dans QGIS.
